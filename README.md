# LuatOS For ESP32

## 1、简介

尝试给ESP32适配一个[LuatOS](https://gitee.com/openLuat/LuatOS)，只是为了学习。希望有更多的朋友来一起贡献。

目前基于ESP-IDF 4.2 release版本

## 2、参与方式

1、vscode安装espidf插件

2、git clone https://gitee.com/dreamcmi/LuatOS-ESP32

3、用vscode打开文件夹，添加/修改/删除

4、提个pr呗
